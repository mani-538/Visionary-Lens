# Visionary Lens - Demo Walkthrough Script

## 1. Introduction (30 Seconds)
"Hello Examiners. This is Visionary Lens. To demonstrate its accessibility, I will be using the application eyes-free, relying on voice feedback and gesture-based navigation."

## 2. Authentication Flow (1 Minute)
- **Step:** Open the App.
- **Voice Feedback:** "Login Screen. Double tap the bottom to register."
- **Action:** Narrate the Login process. "The app uses high-contrast text fields so low-vision users can see them, and TTS for zero-vision users."

## 3. Main Dashboard (30 Seconds)
- **Voice Feedback:** "Welcome to Visionary Lens. Main Menu."
- **Action:** Tap the buttons to show voice feedback. "Every button describes itself aloud. I'll select 'Scan Document' now."

## 4. Scanning & OCR (2 Minutes)
- **Action:** Open Camera View.
- **Voice Feedback:** "Camera Ready. Hold steady."
- **Action:** Simulate a capture.
- **Voice Feedback:** "Image captured accurately. Processing now... Text extracted successfully."
- **Action:** Let the app read out the text. "The app uses Gemini Vision to not only read text but understand it."

## 5. Ask My Notes Feature (1 Minute)
- **Action:** Long-press the screen.
- **Voice Feedback:** "Ask My Notes. Listening."
- **Action:** Ask: "Summarize this page."
- **Voice Feedback:** "Summary... [AI Generated Text]."
- **Action:** Ask: "What are the key formulas here?"
- **Voice Feedback:** "Formulas identified... [AI Generated Text]."

## 6. Study Assistant (1 Minute)
- **Action:** Navigate to Study Assistant.
- **Action:** Select "Generate Quiz".
- **Voice Feedback:** "Quiz generated. Question 1... [AI Generated Question]."
- **Action:** Explain how this helps students prepare for exams independently.

## 7. Conclusion (30 Seconds)
"That concludes the demo. As you saw, the entire flow from scanning a textbook to interacting with its content was done using voice and inclusive gestures. Thank you."
